# ReadtheDocs  
Sphinx + GitHub + ReadtheDocs 作为文档写作工具。  
* Sphinx 生成文档  
* GitHub 托管文档  
* 再导入到 ReadtheDocs  
## 安装sphinx相关依赖  
pip install  
sphinx、sphinx_rtd_theme、recommonmark（使sphinx支持md文件生成html）、jieba（使sphin支持中文搜索）  
## sphinx-quickstart  
在空文件夹目录内cmd
